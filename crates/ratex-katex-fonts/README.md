# ratex-katex-fonts

Embeds the KaTeX `.ttf` files used by RaTeX for optional `embed-fonts` builds (`ratex-svg`, `ratex-render`). Font files ship inside this crate so crates.io releases compile without a monorepo `fonts/` directory.
